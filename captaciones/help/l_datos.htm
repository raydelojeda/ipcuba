Listado de captaciones municipales

Esta página muestra una lista a nivel municipal de todas las captaciones realizadas en el municipio al que pertenece el usuario que está autenticado en este momento.

Iconos de la Barra de Herramientas

  • Nuevo: Pulse el icono ‘Nuevo’ para ingresar nuevos datos.
  • Editar: Pulse el icono ‘Editar’ para modificar la información.
  • Borrar: Pulse el icono ‘Borrar’ para eliminar la información de una fila.
  • Faltantes: Pulse el icono ‘Faltantes’ para mostrar e insertar las captaciones faltantes.
  • Imprimir: Pulse el icono ‘Imprimir’ para imprimir los datos mostrados.
  • Ayuda: Pulse el icono ‘Ayuda’ para ver esta página

Paginador y Menú "Ver #"

En la parte superior derecha del listado aparece un paginador que permite navegar a las siguientes páginas según la cantidad de filas que el menú "Ver #" muestre.

Filtro y Menú "Seleccionar"

Permite acelerar la búsqueda en el listado reduciendo el número de datos mostrados, seleccionando aquellos que tengan ciertas condiciones. Puede filtrar por palabra clave o parte de ella sin importar mayúsculas o minúsculas. Elija en el menú "Seleccionar" un nombre de columna para comenzar la búsqueda.

Encabezado de las Columnas

Esta fila muestra el nombre de las columnas. Pulse el encabezado para reorganizar la columna en orden ascendente o descendente según convenga.

Descripción de las Columnas

  • No: Número consecutivo de filas.
  • Mercado: Esta columna muestra el mercado de la captación.
  • Variedad: Esta columna muestra la variedad de la captación.
  • Establecimiento: Esta columna muestra el establecimiento de la captación.
  • Precio: Esta columna muestra el precio de la captación.
  • Usuario: Esta columna muestra el nombre de conexión del usuario que digitó la captación.
  • Cód. DPA: Esta columna muestra el código de la DPA de la captación.
  • Fecha digitada: Esta columna muestra la fecha de digitación de la captación.
  • : Casilla para marcar todos. (Esta opción es para borrar todas las captaciones)
