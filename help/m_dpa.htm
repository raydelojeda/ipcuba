Control DPA: Modificar

En esta página puede modificar los datos de la DPA, seleccionada.

Iconos de la Barra de Herramientas

  • Guardar: Pulse el icono ‘Guardar’ para guardar los cambios de los datos.
  • Cancelar: Pulse el icono ‘Cancelar’ para volver a la página ‘Administración DPA’.
  • Ayuda: Pulse el icono ‘Ayuda’ para ver esta página.

Descripción de las Cajas de texto

  • Código DPA: Muestra el código de la DPA.
  • Provincia o Municipio: Muestra el nombre de una provincia o un municipio.
