Captación de precios a nivel municipal

En esta página puede ingresar el precio promedio y fecha, del producto.

Iconos de la Barra de Herramientas

  • Guardar: Pulse el icono ‘Guardar’ para guardar los datos del producto.
  • Cancelar: Pulse el icono ‘Cancelar’ para volver a la página ‘Listado a nivel municipal’.
  • Ayuda: Pulse el icono ‘Ayuda’ para ver esta página.

Descripción de las Cajas de texto

  • Mercado: Elija el mercado de la captación.
  • Código producto: Elija el código-producto.
  • Precio mínimo: Precio establecido.
  • Precio: Escriba el precio promedio del producto.
  • Precio máximo: Precio establecido.
  • Provincia: Elija la provincia.
  • Municipio: Elija el municipio.
  • Vendedor: Escriba el nombre del vendedor.
  • Establecimiento: Escriba el nombre del establecimiento.
