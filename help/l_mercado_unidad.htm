Administración mercado-unidad

Esta página muestra la lista de los mercados y las unidades de medida. Puede añadir, eliminar o editar los mercados y las unidades de medida.

Iconos de la Barra de Herramientas

  • Nuevo: Pulse el icono ‘Nuevo’ para ingresar nuevos datos.
  • Editar: Pulse el icono ‘Editar’ para modificar la información.
  • Borrar: Pulse el icono ‘Borrar’ para eliminar la información de una fila.
  • Ayuda: Pulse el icono ‘Ayuda’ para ver esta página

Debe elejir la pestaña ‘Mercados’ para crear, modificar o borrar, los mercados.

Debe elejir la pestaña ‘Unidad Medida’ para crear, modificar o borrar, la unidad de medida.

Encabezado de las Columnas

  • Esta fila muestra el nombre de las columnas.

Descripción de las Columnas

  • Mercado: Muestra el mercado de la captación.
  • Unidad Medida: Muestra la unidad de medida.
  • Casilla de Selección: Pulse esta casilla para seleccionar los datos de una fila
