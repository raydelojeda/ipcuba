Control del vendedor-establecimiento: Insertar

En esta página puede crear un nuevo vendedor y un nuevo establecimiento.

Iconos de la Barra de Herramientas

  • Guardar: Pulse el icono ‘Guardar’ para guardar los datos.
  • Cancelar: Pulse el icono ‘Cancelar’ para volver a la página ‘Administración vendedor-establecimiento’.
  • Ayuda: Pulse el icono ‘Ayuda’ para ver esta página.

Descripción de las Cajas de texto

  • Vendedor: Escriba el nombre del vendedor.
  • Establecimiento: Escriba el nombre del establecimiento.
