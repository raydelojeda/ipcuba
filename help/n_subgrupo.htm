Control de subgrupo: Isertar

En esta página puede introducir los datos del nuevo subgrupo.

Iconos de la Barra de Herramientas

  • Guardar: Pulse el icono ‘Guardar’ para guardar los datos del subgrupo.
  • Cancelar: Pulse el icono ‘Cancelar’ para volver a la página ‘Administración de subgrupo’.
  • Ayuda: Pulse el icono ‘Ayuda’ para ver esta página.

Descripción de las Cajas de texto

  • Grupo: Elija el grupo.
  • Código del subgrupo: Escriba el código del subgrupo.
  • Subgrupo: Escriba el nombre del subgrupo.
