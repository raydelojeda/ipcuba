Control del Producto: Modificar

En esta página se muestra el producto seleccionado. Puede modificar los datos.

Iconos de la Barra de Herramientas

  • Guardar: Pulse el icono ‘Guardar’ para guardar los cambios de los datos.
  • Cancelar: Pulse el icono ‘Cancelar’ para volver a la página ‘Administración de producto’.
  • Ayuda: Pulse el icono ‘Ayuda’ para ver esta página.

Descripción de las Cajas de texto

  • Artículo: Muestra el artículo.
  • Código del producto: Muestra el código del producto.
  • Producto:Muestra el nombre del producto.
