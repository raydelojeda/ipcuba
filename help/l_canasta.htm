Nomenclador Canasta Básica

Esta página muestra una lista de los Capítulos, Divisiones, Grupos, Subgrupos, Artículos y Productos de la canasta básica del año base. Puede añadir, eliminar o modificar los Capítulos, Divisiones, Grupos, Subgrupos, Artículos y Productos.

Iconos de la Barra de Herramientas

  • Imprimir: Pulse el icono ‘Imprimir’ para imprimir los datos mostrados.
  • Ayuda: Pulse el icono ‘Ayuda’ para ver esta página.

Paginador

A la derecha y arriba del listado se muestra un paginador que permite navegar a las siguientes páginas.

Encabezado de las Columnas

  • Esta fila muestra el nombre de las columnas. Pulse el encabezado para reorganizar la columna en orden diferente.
Descripción de las Columnas
  • Capítulo: Esta columna muestra el Capítulo.
  • División: Esta columna muestra la División.
  • Grupo: Esta columna muestra el Grupo.
  • Subgrupo: Esta columna muestra el Subgrupo.
  • Artículo: Esta columna muestra el Artículo.
  • Producto: Esta columna muestra el producto.
