Producto del año base: Isertar

En esta página puede seleccionar un producto-mercado-unidad de medida, y modificar el precio promedio, valor base, fecha, precio mínimo y precio máximo del producto.

Iconos de la Barra de Herramientas

  • Guardar: Pulse el icono ‘Guardar’ para guardar los datos del producto.
  • Cancelar: Pulse el icono ‘Cancelar’ para volver a la página ‘Listado de productos del año base’.
  • Ayuda: Pulse el icono ‘Ayuda’ para ver esta página.

Descripción de las Cajas de texto

  • Producto: Elija el producto.
  • Mercado: Elija el mercado de la captación.
  • Unidad de medida: Elija la unidad de medida del producto.
  • Precio: Escriba el precio del producto.
  • Valor Base: Escriba el valor base del producto.
  • Fecha: Escriba la fecha del producto.
  • Precio Mínimo: Escriba el precio mínimo del producto.
  • Precio Máximo: Escriba el precio máximo del producto.
