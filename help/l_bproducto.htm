Listado de productos del año base

Esta página muestra una lista de los productos, mercado, intervalo de precio (pmínimo-pmáximo), unidad de medida, valor, precio promedio, fecha y peso (ponderación), del año base. Puede editar el intervalo de precio, unidad de medida, valor base, precio, fecha y peso (ponderación) de los productos.

Iconos de la Barra de Herramientas

  • Nuevo: Pulse el icono ‘Nuevo’ para ingresar nuevos datos.
  • Editar: Pulse el icono ‘Editar’ para modificar la información.
  • Borrar: Pulse el icono ‘Borrar’ para eliminar la información de una.
  • Imprimir: Pulse el icono ‘Imprimir’ para imprimir los datos mostrados.
  • Ayuda: Pulse el icono ‘Ayuda’ para ver esta página

Paginador


A la derecha y arriba del listado se muestra un paginador que permite navegar a las siguientes páginas.

Filtro y Menú "Seleccionar"

Permite acelerar la búsqueda en el listado reduciendo el número de datos mostrados, seleccionando aquellos que tengan ciertas condiciones. Puede filtrar por palabra clave o parte de ella. Elija en el menú seleccionar un nombre de columna para comenzar la búsqueda.

Encabezado de las Columnas

  • Esta fila muestra el nombre de las columnas. Pulse el encabezado para reorganizar la columna en orden diferente.

Descripción de las Columnas

  • Mercado: Esta columna muestra el mercado de la captación.
  • Producto: Esta columna muestra el producto.
  • Mínimo: Esta columna muestra el precio mínimo del producto.
  • Máximo: Esta columna muestra el precio máximo del producto.
  • Unidad: Esta columna muestra la unidad de medida del producto.
  • Valor: Esta columna muestra el valor del producto.
  • Precio: Esta columna muestra el precio promedio del producto.
  • Fecha: Esta columna muestra la fecha del año base.
  • Peso: Esta columna muestra la ponderación del producto.
  • Casilla de Selección: Pulse esta casilla para seleccionar los datos de una fila.
