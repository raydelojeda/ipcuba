Control de artículo: Modificar

En esta página se muestra el artículo seleccionado. Puede modificar los datos.

Iconos de la Barra de Herramientas

  • Guardar: Pulse el icono ‘Guardar’ para guardar los cambios de los datos.
  • Cancelar: Pulse el icono ‘Cancelar’ para volver a la página ‘Administración de artículo’.
  • Ayuda: Pulse el icono ‘Ayuda’ para ver esta página.

Descripción de las Cajas de texto

  • Subgrupo: Muestra el subgrupo.
  • Código del Artículo: Muestra el código del artículo.
  • Artículo: Muestra el nombre del artículo.
