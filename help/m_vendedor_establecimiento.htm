Control del vendedor: Modificar

Esta página muestra el vendedor seleccionado. Puede modificar el nombre.

Iconos de la Barra de Herramientas

  • Guardar: Pulse el icono ‘Guardar’ para guardar los cambios de los datos.
  • Cancelar: Pulse el icono ‘Cancelar’ para volver a la página ‘Administración vendedor-establecimiento’.
  • Ayuda: Pulse el icono ‘Ayuda’ para ver esta página.

Descripción de la caja de texto

  • Vendedor: Muestra el nombre del vendedor seleccionado.
