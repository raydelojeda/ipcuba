Producto del año base: Modificar

En esta página puede modificar el precio promedio, valor base, fecha, precio mínimo y precio máximo del producto seleccionado.

Iconos de la Barra de Herramientas

  • Guardar: Pulse el icono ‘Guardar’ para guardar los cambios de los datos.
  • Cancelar: Pulse el icono ‘Cancelar’ para volver a la página ‘Listado de productos del año base’.
  • Ayuda: Pulse el icono ‘Ayuda’ para ver esta página.

Descripción de las Cajas de texto

  • Precio: Muestra el precio del producto.
  • Valor Base: Muestra el valor base del producto.
  • Fecha: Muestra la fecha del año base.
  • Precio Mínimo: Muestra el precio mínimo del producto.
  • Precio Máximo: Muestra el precio máximo del producto.
  • Unidad de medida: Muestra la unidad de medida del producto.
