Control del producto: Isertar

En esta página puede introducir los datos del producto nuevo.

Iconos de la Barra de Herramientas

  • Guardar: Pulse el icono ‘Guardar’ para guardar los datos del producto.
  • Cancelar: Pulse el icono ‘Cancelar’ para volver a la página ‘Administración de producto’.
  • Ayuda: Pulse el icono ‘Ayuda’ para ver esta página.

Descripción de las Cajas de texto

  • Artículo: Elija el artículo.
  • Código del producto: Escriba el código del producto.
  • Producto: Escriba el nombre del producto.
