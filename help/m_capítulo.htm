Control de Capítulo: Modificar

En esta página puede modificar los datos del capítulo.

Iconos de la Barra de Herramientas

  • Guardar: Pulse el icono ‘Guardar’ para guardar los cambios de los datos.
  • Cancelar: Pulse el icono ‘Cancelar’ para volver a la página ‘Administración de capítulo’.
  • Ayuda: Pulse el icono ‘Ayuda’ para ver esta página.

Descripción de las Cajas de texto

  • Código del Capítulo: Muestra el código del capítulo.
  • Capítulo: Muestra el nombre del capítulo.
