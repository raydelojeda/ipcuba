Control del mercado-unidad medida: Isertar

En esta página puede introducir los datos del nuevo mercado y/o la nueva unidad de medida.

Iconos de la Barra de Herramientas

  • Guardar: Pulse el icono ‘Guardar’ para guardar los datos.
  • Cancelar: Pulse el icono ‘Cancelar’ para volver a la página ‘Administración mercado-unidad’.
  • Ayuda: Pulse el icono ‘Ayuda’ para ver esta página.

Descripción de las Cajas de texto

  • Mercado: Escriba el nombre del mercado.
  • Unidad medida: Escriba la unidad de medida.
