Control del artículo: Isertar

En esta página puede introducir los datos del nuevo artículo.

Iconos de la Barra de Herramientas

  • Guardar: Pulse el icono ‘Guardar’ para guardar los datos del artículo.
  • Cancelar: Pulse el icono ‘Cancelar’ para volver a la página ‘Administración del artículo’.
  • Ayuda: Pulse el icono ‘Ayuda’ para ver esta página.

Descripción de las Cajas de texto

  • Subgrupo: Elija el subgrupo.
  • Código del artículo: Escriba el código del artículo.
  • Artículo: Escriba el nombre del artículo.
